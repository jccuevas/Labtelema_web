Operador de propagación - (...)

El operador de propagación (spread operator), representado por tres puntos (...), permite la expansión de una expresión en situaciones donde se esperan múltiples argumentos como en las llamadas a funciones, o múltiples elementos como en arrays literales.

function listar (...elementos){
  elementos.forEach((element,index) => {
    document.write(`<p>${index} ${element}</p>`);
  });
}
