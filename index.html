Labtelema

Servidor de prácticas del Área de Ingeniería Telemática del Departamento de Ingeniería de Telecomunicación.

Soporte para asignaturas

Este servidor está pensado para dar soporte a las asiganturas del área de Ingeniería Telemática en los grados y másteres en los que imparte docencia.

Algunas de las asignaturas que usan este servidor son:

  • Protocolos de transporte
  • Servicios Móviles
  • Servicios y aplicaciones telemáticas
  • Realidad Virtual

Servicios disponibles

  • Servidor de correo SMTP, POP3 e IMAP.
  • Servidor de web Apache con soporte para PHP, HTTPS.
  • Servidor Node.js.
  • Gestor de paquetes npm.
  • Motor de bases de datos MySQL.
  • Motor de bases de datos MongoDB.
